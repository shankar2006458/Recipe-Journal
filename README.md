# 🍽️ Recipe Journal

Recipe Journal is a simple and elegant recipe finder web app built using HTML, CSS, JavaScript, and TheMealDB API.

## Features

- 🔍 Search recipes by name
- 🎲 Get random recipes
- ❤️ Save favorite recipes
- 📖 View ingredients and instructions
- 🎥 Watch recipe tutorials on YouTube
- 🕒 Search history
- 📱 Fully responsive design
- 💾 Local storage support

## Technologies Used

- HTML5
- CSS3
- JavaScript
- TheMealDB API
- Local Storage

## How to Use

1. Search for a recipe.
2. Browse recipe results.
3. Click **View Recipe** for details.
4. Save recipes to **My Cookbook**.
5. Watch tutorials on YouTube.
